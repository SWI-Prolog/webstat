The execution profiler gathers statistical information about the execution of your program. These statistics can be used to analyse performance. There are two ways to record profiling data:

  1. Run ?- profile(Goal). or
  2. Use the and buttons while your program is running.

After recording, the results can be examined using the button.

  • Erase all recorded profile data.
  • Start recording a (CPU) profile. While recording, the button is red.
  • Stop recording profile information.
  • Show the collected data as a list of used predicates sorted by the sampled CPU time used.
